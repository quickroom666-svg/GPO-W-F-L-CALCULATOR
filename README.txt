# GPO Trade — Grid Calculator (Static Site)

This build was generated from your spreadsheet and is **public read‑only by default**.
Open with `?admin=1` to unlock admin tools (local only). To make changes permanent for everyone:

1. Use **Import Catalog** / **Upload Image** in admin mode (optional, local preview).
2. Click **Export Catalog** to download `catalog.json`.
3. Replace `catalog.json` in your host (Netlify/GitHub Pages) and upload any images to `/assets/`.
4. Redeploy the site.

Files:
- `index.html` — the app
- `catalog.json` — full item list with values (from your spreadsheet), images can be `assets/<file>` or external URLs
- `assets/placeholder.png` — default image

Deploy:
- GitHub Pages: put all files in a repo and enable Pages (root).
- Netlify: app.netlify.com/drop → drop the folder.
- Cloudflare Pages: Create Project → “Upload assets”.
